# DarkSwap — Jupiter DEX Integration (`darkswap/`)

Jupiter DEX integration examples and reference implementations for Solana swaps on the ZOLana platform.

## Contents

| Directory | Description |
|-----------|-------------|
| `jupiter-core-example-main/` | Example usage of the Jupiter Core AMM SDK (Rust) |
| `jupiter-quote-api-node-main/` | Node.js API client for the Jupiter Quote API (fetch swap quotes) |
| `jupiter-swap-api-client-main/` | Node.js client for the Jupiter Swap API (execute swaps) |

## Integration

These examples demonstrate how to integrate Jupiter — Solana's leading DEX aggregator — for optimal swap routing across all Solana liquidity sources. Used by the Dark Protocol for private swaps.

## Usage

Each subdirectory is a standalone project. See the individual README files within each for setup and usage instructions.
