# DarkSwap — Jupiter DEX Integration (`darkswap/`)

Jupiter DEX integration examples and reference implementations for Solana swaps on the ZOLana platform.

These subdirectories are reference/example projects and are not part of the
supported public release surface.

## Contents

| Directory | Description |
|-----------|-------------|
| `jupiter-core-example-main/` | Example usage of the Jupiter Core AMM SDK (Rust) |
| `jupiter-quote-api-node-main/` | Reference Node.js client example for the Jupiter Quote API |
| `jupiter-swap-api-client-main/` | Reference Node.js client example for the Jupiter Swap API |

## Integration

These examples demonstrate how to integrate Jupiter for optimal swap routing
across Solana liquidity sources. Production swap behavior should be implemented
through the maintained Dark Protocol and wallet surfaces, not by shipping these
sample projects directly.

## Usage

Each subdirectory is a standalone project. See the individual README files within each for setup and usage instructions.
