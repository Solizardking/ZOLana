# Dark Protocol Client Components

This directory is an archived set of React component examples for Dark
Protocol integrations.

It is not part of the supported public release surface and is not maintained as
an independently installable package.

Use these maintained surfaces instead:

- `dark-wallet/` for the supported browser wallet UI
- `dark-protocol/sdk/typescript/` for the supported TypeScript SDK

The component snippets under `src/components/` remain here as reference code
only.
