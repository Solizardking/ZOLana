# ZOLana — Zcash Utilities (`zcutil/`)

Utility scripts for building, testing, and managing the Zcash node.

## Contents

| File / Directory | Description |
|-----------------|-------------|
| `build.sh` | Build script for the Zcash node |
| `build-debian-package.sh` | Build Debian/Ubuntu .deb package |
| `clean.sh` | Clean build artifacts |
| `cleanup-tags.sh` | Clean up git tags |
| `distclean.sh` | Deep clean (autotools distclean) |
| `fetch-params.sh` | Download Zcash proving/verifying parameters |
| `make-release.py` | Automated release creation script |
| `release-notes.py` | Release notes generation |
| `afl/` | American Fuzzy Lop fuzz testing configuration |
| `libfuzzer/` | libFuzzer fuzz testing configuration |
